Project : SAP Chatbot

Question answering from pptx files
Description :
The notebook "answering_pptx.ipynb" serves the purpose of extracting data from multiple PowerPoint (pptx) files. It utilizes a question-answering pipeline based on the Transformers library, enabling users to pose questions and receive answers based on the content extracted from the pptx files. The notebook facilitates the retrieval of relevant information from the pptx documents, aiding users in finding answers to specific queries.

Features :
  • Data Extraction: The notebook extracts text and relevant information from multiple pptx files.
  • Question-Answering Pipeline: Utilizes the Transformers library to implement a question-answering pipeline.
  • Interactive Querying: Users can input questions and receive answers based on the content of the pptx files.
  • Document Identification: Provides the name of the pptx document containing the answer to the user's query.
  • NLP Processing: Implements natural language processing techniques to understand and respond to user queries effectively.
Suggesting similar top 5 questions
Description :
The notebook "Moteur.ipynb" serves as a data scraping engine that extracts information from the FAQ website of SAP using BeautifulSoup. It employs cosine similarity to respond to user queries by identifying the top 5 similar questions from the website. The notebook enhances user engagement by providing relevant and informative responses sourced from the SAP FAQ database.




Features :
  • Web Scraping: Utilizes BeautifulSoup to scrape data from the FAQ website of SAP.
  • Cosine Similarity: Computes the similarity between user queries and FAQ entries using cosine similarity.
  • Top Similar Questions: Identifies and presents the top 5 questions most similar to the user query.
  • Efficient Response Generation: Generates responses promptly by leveraging pre-existing FAQ content.
  • User Interaction: Allows users to interactively pose questions and receive informative answers sourced from the SAP FAQ database.
Keywords : Python Jupyter Transformers
